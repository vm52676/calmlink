# CalmLink Suite

A collection of three gentle, student-focused web apps hosted together from one GitHub Pages repo.

## Apps

| App | Path | Age Range | Description |
|---|---|---|---|
| 🧘 CalmLink | `/` | All ages | Student wellbeing micro-app with mood check-ins and grounding exercises |
| 🚀 Driftvoid | `/driftvoid/` | 11+ | Exploration and adventure |
| 🌲 Little Forest | `/littleforest/` | 5+ | 10-minute daily 3D forest sanctuary |

## Live URLs (after GitHub Pages is enabled)

```
https://yourusername.github.io/calmlink/
https://yourusername.github.io/calmlink/driftvoid/
https://yourusername.github.io/calmlink/littleforest/
```

## File Structure

```
calmlink/
├── index.html          ← CalmLink main app
├── driftvoid/
│   └── index.html      ← Driftvoid app
├── littleforest/
│   └── index.html      ← Little Forest game
└── README.md
```

## Deploying

1. Push all files to a public GitHub repository
2. Go to Settings → Pages → Source → select `main` branch → Save
3. Site goes live at `https://yourusername.github.io/reponame/`

## Notes

- Replace `ca-pub-XXXXXXXXXXXXXXXX` and the ad slot ID in `index.html` with your real Google AdSense IDs
- Paste your full Driftvoid app content into `driftvoid/index.html` where indicated
